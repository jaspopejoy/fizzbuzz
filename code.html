                            //get the fizz and buzz values from the user
                            function getValues() {

                                //get the user values from the document
                                let fizzValue = document.getElementById("fizzValue").value;
                                let buzzValue = document.getElementById("buzzValue").value;

                                //parse for numbers
                                fizzValue = parseInt(fizzValue);
                                buzzValue = parseInt(buzzValue);

                                //check to make sure they are integers
                                if (Number.isInteger(fizzValue) && Number.isInteger(buzzValue)) {

                                    //call fizzBuzz
                                    let fbArray = fizzBuzzB(fizzValue, buzzValue);
                                    //call displayData and write values to the screen
                                    displayData(fbArray);
                                } else {
                                    alert("You must enter an integer!")
                                }

                            }

                            //do fizzBuzz
                            //traditional solve for FizzBuzz using a for loop
                            function fizzBuzz(fizzValue, buzzValue) {

                                let returnArray = [];

                                //loop from 1 to 100
                                for (let i = 1; i <= 100; i++) {

                                    //check to see if value is divisible by 3 and 5
                                    //check to see if divisible by buzz value (5)
                                    //check to see if divisible by fizz value (3)
                                    if (i % fizzValue == 0 && i % buzzValue == 0) {
                                        returnArray.push('FizzBuzz');
                                    } else if (i % fizzValue == 0) {
                                        returnArray.push('Fizz');
                                    } else if (i % buzzValue == 0) {
                                        returnArray.push('Buzz');
                                    } else {
                                        returnArray.push(i);
                                    }
                                }
                                return returnArray

                            }

                            // another way to solve for FizzBuzz using a switch
                            function fizzBuzzB(fizzValue, buzzValue) {
                                let returnArray = [];
                                let Fizz = false;
                                let Buzz = false;

                                for (let i = 1; i <= 100; i++) {

                                    Fizz = i % fizzValue == 0;
                                    Buzz = i % buzzValue == 0;

                                    switch (true){
                                        case Fizz && Buzz: {
                                            returnArray.push("FizzBuzz");
                                            break;
                                        }
                                        case Fizz: {
                                            returnArray.push("Fizz");
                                            break;
                                        }
                                        case Buzz: {
                                            returnArray.push("Buzz");
                                            break;
                                        }
                                        default: {
                                            returnArray.push(i);
                                            break;
                                        }
                                    }
                                }
                                return returnArray
                            }

                            function fizzBuzzC(fizzValue, buzzValue) {
                                let returnArray = [];
                                for (let i = 1; i<= 100; i++) {
                                    let value = ((i % fizzValue == 0 ? 'Fizz' : '') + (i % buzzValue == 0 ? 'Buzz' : '') || i);

The Code is structured in Three functions.

getValues

getValues accepts(gets) the user input from the page using getElementById and stores that input in the fizzValue and BuzzValue variables. The values are then passed to the fizzBuzz function and stores the resulting array in the fbArray variable. This variable is then passed to the displayData function to be printed out to the user.

fizzBuzz

The fizzBuzz function is one of three ways to solve this challenge. This function starts by creating an empty array named fbArray. Then a for loop is used to cycle through the numbers from 1 to 100. An if statement is used to check if the number is a multiple of both numbers entered by the user. If they are, FizzBuzz is pushed into the array.

Two else if statements are then used to check if the number is a multiple of the first value and pushes Fizz, or a multiple of the second value and pushes buzz. If the number is not a multiple of either value, an else statement pushes the number itself into the array.

fizzBuzzB

This is the second way to solve this challenge. fizzBuzzB begins by creating the array fbArray. The variables fizz and buzz are then set to false. A for loop is created to cycle throug the numbers from 1 to 100, and the numbers are evaluated against the fizzValue and buzzValue variables. A switch statement is then used to determine which statments are true using case clauses. The first clause checks the values of fizz and buzz to see if they both equal true. If so, FizzBuzz is pushed into the array. The second clause checks to see if Fizz equals true and the third if Buzz is true and pushes either fizz or buzz into the array. Finally, if all three cases equal false, the default will push a number into the array.

fizzBuzzC

The third way to solve this challenge is fizzBuzzC. FizzbuzzC uses a tienary operator to loop over the numbers 1 to 100 and evaluate if fizzValue and buzzValue are multiples of the numbers entered by the user.

displayData

The displayData function creates HTML to insert into a display table. By using a template element in the app.html table rows are created with a for loop. Using the importNode function and querySelectorAll we can create the HTML code with the template element. The classList and the textContent properties are chanved at the same time to create the tableRow HTML and assign a class that can be targeted by our CSS.
